# Resolver - приложение для решения проблем отдела кафедры АСУ в институте.

## Стек:

### Frontend:
* ReactJS, Hooks, React-Router
* Redux
* Axios
* SASS(SCSS)
* HowlerJS
* Material UI

### BackEnd:
* NodeJS
* ExpressJS
* MongoDB / Mongoose / NoSQL
* BcryptJS
* Socket.io

## Ссылка на рабочее приложение: https://resolver-chat-app.herokuapp.com/
